Hva er Css?

Definisjon, funksjonalitet

du kan si at Css er designet på bilen.
CSS er et stylingspråk som brukes til å gjøre nettsidene mer presentable.CSS er et stylingspråk som brukes til å beskrive utseendet og formateringen av et dokument.HTML, CSS og JavaScript betraktes som de viktigste tre grunnleggende språkene for webutvikling. Med CSS kan man lage animasjoner, som minecraft-animasjonen på bildet.
CSS-stilsett består ganske enkelt av en liste med stilregler. En stilregel har to deler: selektoren som forteller hvor den gjelder og deklarasjons-listen som forteller hvordan formateringen skal være.

Eksempel

 <p style="color:red">Nå har jeg stylet teksten med rød farge</p>
 <p style="color:pink">Nå har jeg stylet teksten med rosa farge</p>

Denne setningen kan skifte farge!

Eksempel

 <p style="text-align: left">Nå plasserer jeg teksten til venstre</p>
 <p style="text-align: center">Nå sentrerer jeg teksten</p>

Denne teksten er flyttbar!
